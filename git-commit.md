# Git Commit

El comando git commit es una forma de confirmar el contenido presente en la "Staging Area" o "Area de preparación". Se cargan el conjunto de archivos añadidos junto con un mensaje que debe ser descriptivo de la serie de cambios realizados.

El contenido que será confirmado mediante git commit se puede gestionar previamente haciendo un uso selectivo del git add, o listando los archivos como argumentos del comando.
