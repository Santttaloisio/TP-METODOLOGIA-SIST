# Git Add

El comando ```git add``` tiene como función agregar los archivos nuevos o modificados a la "Staging Area" o área de preparación, de la cual se obtendrá el conjunto de elementos que entrarán en el siguiente commit.
También es posible agregar partes de modificaciones a un archivo con el comando ```git add -p```. 
Se puede ejecutar tantas veces como sea necesario antes del commit, ya que solo agrega contenidos al índice.
